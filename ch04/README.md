# CH04 本地存储管理

本章讲解 磁盘分区、逻辑卷管理 和 文件系统 的管理。

* 磁盘分区
  * MBR 和 GPT 分区表
  * `fdisk` 和 `gdisk` 分区工具
* LVM
  * 物理卷、卷组、逻辑卷
  * 创建 物理卷、卷组、逻辑卷
  * 查看 物理卷、卷组、逻辑卷
  * 扩展和收缩 卷组、逻辑卷 
* 文件系统
  * 使用文件系统的一般方法
  * 创建文件系统
  * 挂装/卸装 文件系统
  * 在系统启动时挂装文件系统
  * 挂装和使用移动存储介质
  * 挂装和使用映像文件
  * 文件系统的检查和修复
  * 文件系统的扩展和收缩
  * 使用交换分区和交换文件
  * 设置磁盘限额
